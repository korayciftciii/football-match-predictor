# Installation instructions:
# 1. Create virtual environment: python -m venv venv
# 2. Activate virtual environment: source venv/bin/activate (Linux/Mac) or venv\Scripts\activate (Windows)
# 3. Install development dependencies: pip install -r requirements-dev.txt
# 4. Install pre-commit hooks: pre-commit install
# 5. Run tests: pytest
# 6. Run linting: flake8 app/ tests/
# 7. Format code: black app/ tests/
# 8. Sort imports: isort app/ tests/
# 9. Type check: mypy app/

# Common development commands:
# - Run all quality checks: pre-commit run --all-files
# - Run tests with coverage: pytest --cov=app --cov-report=html
# - Generate documentation: mkdocs serve
# - Profile application: python -m cProfile -o profile.stats -m app.main
# - Security scan: bandit -r app/
# - Dependency vulnerability check: safety check

# IDE Configuration:
# - VS Code: Install Python extension and configure settings.json
# - PyCharm: Configure interpreter and enable code quality tools
# - Vim/Neovim: Install appropriate plugins for Python development

# Troubleshooting:
# - If installation fails, try upgrading pip: pip install --upgrade pip
# - For M1 Mac issues, use: pip install --no-use-pep517 <package>
# - For Windows issues, ensure Visual Studio Build Tools are installed
